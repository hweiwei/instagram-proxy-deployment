## 核心组件

### 1. Xray Core

| 功能 | 说明 |
|------|------|
| **多协议支持** | VLESS、VMess、Trojan、WebSocket |
| **TLS 伪装** | 模拟 Chrome/Firefox/Safari 指纹 |
| **流量转发** | 高性能流量转发 |
| **路由规则** | 智能分流，Instagram 直连 |
| **连接管理** | 多路复用，连接池 |

### 2. Nginx 反向代理

| 功能 | 说明 |
|------|------|
| **SSL 终结** | 处理 TLS 握手 |
| **WebSocket 代理** | 支持 VMess WebSocket |
| **请求伪装** | 伪装成正常 HTTPS 流量 |
| **负载均衡** | 多后端均衡 |
| **缓存** | 静态资源缓存 |
| **安全头** | 添加安全响应头 |

### 3. 防火墙 (iptables)

| 功能 | 说明 |
|------|------|
| **端口控制** | 仅开放必要端口 |
| **连接限制** | 防止暴力破解 |
| **速率限制** | 防止 DDoS |
| **GeoIP** | 可选地域限制 |

### 4. BBR 加速

| 功能 | 说明 |
|------|------|
| **拥塞控制** | BBR 算法 |
| **快速打开** | TCP Fast Open |
| **MTU探测** | 自动MTU调整 |

## 数据流向

```
┌─────────────────────────────────────────────────────────────────────────────────────┐
│                              完整请求流程                                             │
└─────────────────────────────────────────────────────────────────────────────────────┘

用户操作 (Instagram App/浏览器)
        │
        │ 1. 发起 HTTPS 请求
        ▼
手机/电脑代理客户端
        │
        │ 2. 加密并发送到服务器
        ▼
Vultr 服务器公网 IP:443/10086/10087
        │
        │ 3. 防火墙检查
        ▼
Nginx 反向代理 (443)
        │
        │ 4. TLS 解密 + 指纹验证
        ▼
Xray Core (10086/10087/10089)
        │
        │ 5. 协议解析 + 路由判断
        ▼
┌───────────────────────────────────────────────────────────────────────┐
│                           路由决策                                       │
├───────────────────────────────────────────────────────────────────────┤
│                                                                       │
│   if (目标域名包含 instagram.com 或 157.240.0.0/16)                    │
│       → 路由到 Instagram 直连                                          │
│                                                                       │
│   else if (目标域名为国内域名 或 GEOIP:CN)                             │
│       → 路由到国内直连 (不走代理)                                      │
│                                                                       │
│   else                                                                 │
│       → 通过代理转发到目标                                              │
│                                                                       │
└───────────────────────────────────────────────────────────────────────┘
        │
        │ 6. 直连模式: Freedom 协议直接连接
        │ 7. 代理模式: 转发到目标服务器
        ▼
Instagram 服务器
        │
        │ 8. 返回响应
        ▼
反向路径: Instagram → Xray → Nginx → 客户端
```

## 安全机制


### 2. 连接优化

| 参数 | 默认值 | 优化值 | 说明 |
|------|--------|--------|------|
| `tcp_fastopen` | 0 | 3 | TCP快速打开 |
| `somaxconn` | 128 | 65535 | 最大连接队列 |
| `file-max` | 系统默认 | 6553600 | 最大文件描述符 |
| `ip_local_port_range` | 32768-60999 | 1024-65535 | 端口范围 |

### 关键监控指标

| 指标 | 告警阈值 | 说明 |
|------|----------|------|
| Xray 进程状态 | down | 服务宕机 |
| CPU 使用率 | > 80% | 系统过载 |
| 内存使用率 | > 85% | 内存不足 |
| 磁盘使用率 | > 90% | 空间不足 |
| 连接数 | > 5000 | 可能被滥用 |
| 带宽使用 | > 90% 带宽上限 | 带宽不足 |
| SSL 证书过期 | < 30天 | 证书即将过期 |
